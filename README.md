# 🐾 My Little Corner

A peaceful space to manage daily life, built with Java Spring Boot.

### ✨ What's Inside?
* **Daily Diary:** Ghi lại những mẩu chuyện nhỏ xinh.
* **Brain Dump:** Nơi lưu giữ những ý tưởng bất chợt.
* **Routine:** Quản lý thói quen 3 buổi sáng - chiều - tối.
* **Academic Kanban:** Theo dõi tiến độ học tập hằng ngày.
* **Media Log:** Lưu lại những bộ phim và cuốn sách tâm đắc.

### 🛠️ Tech Stack
* **Backend:** Java 17, Spring Boot 3.2.4
* **Frontend:** HTML, CSS, Thymeleaf
* **Database:** H2 Database (File-based)
* **Interaction:** Fetch API (Ajax)

### 🚀 Quick Start
1. Clone the project.
2. Open in VS Code or IntelliJ.
3. Run `MyWebApplication.java`.
4. Visit `localhost:8080` (Pass: `123456`).
